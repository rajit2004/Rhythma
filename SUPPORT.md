# Support

Thank you for your interest in Rhythma.

## Where to ask questions

### GitHub Issues

Use GitHub Issues for:

- Bug reports
- Feature requests
- Documentation issues

---

### General questions

Rhythma doesn't yet have a community chat (Discord/Slack) set up. For now,
general questions, setup help, or brainstorming can go in
[GitHub Discussions](https://github.com/ishita2740/Rhythma/discussions)
(if enabled) or as a GitHub Issue tagged `question`.

---

### Security issues

Please **do not** report security vulnerabilities publicly through Issues.

Instead, contact the maintainer directly via GitHub
([@ishita2740](https://github.com/ishita2740)) until a formal
`SECURITY.md` policy is added.

---

### Before asking for help

Please check:

- [README.md](README.md)
